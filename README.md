A Easy Adapter for RecyclerView

You can use gradle
```
repositories {
  mavenCentral()
  google()
}

dependencies {
  implementation 'com.ivy.adapter:EasyAdapter:1.0.0'
}
```

or use maven
```
<dependency>
  <groupId>com.ivy.adapter</groupId>
  <artifactId>EasyAdapter</artifactId>
  <version>1.0.0</version>
  <type>pom</type>
</dependency>
```
